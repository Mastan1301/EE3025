\section{Problem}

    
The command
\begin{lstlisting}
    output_signal = signal.lfilter(b,a,output_signal)
\end{lstlisting}

in Problem 2.3 is executed through following difference equation 
    \begin{align}
        \sum _{m=0}^{M}a\brak{m}y\brak{n-m}=\sum _{k=0}^{N}b\brak{k}x\brak{n-k}
    \label{eq:ee18btech11039_eq1}
    \end{align}
 where input signal is $x(n)$ and output signal is $y(n)$ with intial values all 0. Replace \textbf{signal.filtfilt} with your own routine and verify
  \section{Solution}
  Using the properties of z-transform
  \begin{align}
      {\mathcal {Z}}\{x(n-k)\} = z^{-k}X(z) \\
      {\mathcal {Z}}\{y(n-m)\} = z^{-m}Y(z)
  \end{align}
  where $X(z)$ and $Y(z)$ are the respective z-transforms of $x(n)$ and $y(n)$ respectively.
\newline
Applying z-transform on both sides in \eqref{eq:ee18btech11039_eq1},
\begin{align}
     Y\brak{z} \sum _{m=0}^{M}a\brak{m} z^{-m} = X\brak{z} \sum _{k=0}^{N}b\brak{k}z^{-k}
\end{align}

\begin{align}
\label{eq:Hz_equation}
    H\brak{z} = \frac{Y\brak{z}}{X\brak{z}} = \frac{\sum _{k=0}^{N}b\brak{k} z^{-k}}{\sum _{m=0}^{M}a\brak{m}z^{-m}}
\end{align}   

\section{Implementation in C}
We first generate the \textit{x.dat} file by reading the data from \textit{Sound\_Noise.wav} file and storing it in a dat file. Next, we generate \textit{H.dat} file which is the DFT of the filter's impulse response, that is computed using the filter coefficients b, a. 

The python code for doing this is-

\begin{lstlisting}
codes/generateDat.py
\end{lstlisting}

Then, we read the \textit{x.dat} file to obtain $x[n]$ and compute the DFT of \textit{x}. Then we obtain the output of the filter in frequency domain as-
\begin{align}
    Y\brak{K} = H\brak{K}X\brak{K}
\end{align}

Then we store $Y$ in \textit{Y.dat} file and the time domain signal (computed using ifft()) $y$ in \textit{y.dat} file.

This is done using the below C program.
\begin{lstlisting}
codes/fft.c
\end{lstlisting}

Compile the program using-
\begin{lstlisting}
gcc fft.c -o fft -lm
\end{lstlisting}

Run the program using - 
\begin{lstlisting}
./fft
\end{lstlisting}

Finally, we store the outputs in wav files and plot the two outputs, one done using C and the other using the library functions. The python code for this is - 
\begin{lstlisting}
codes/ee18btech11039.py
\end{lstlisting}

The soundfile obtained using C program is -
